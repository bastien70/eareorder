# eareorder
Implémentation de la fonctionnalité de reindex sur EasyAdmin 3

## Installation

1. `git clone git@github.com:bastien70/eareorder.git`

2. `cd earoder`

3. `composer install`

4. `yarn install`

5. `yarn run build`

6. `php bin/console d:d:c` (après avoir paramétré la base)

7. `php bin/console d:m:m`

8. `symfony serve -d`
